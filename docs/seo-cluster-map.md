## Objective

- unified PIM + DAM

This plan is based on:

This is intent-led keyword research, not a volume export from Ahrefs or Semrush. Use Search Console and a keyword tool later to validate volumes and difficulty before a full publishing sprint.

## Positioning Rules For New SEO Pages

- Keep the category language broad enough to rank and narrow enough to qualify.
- Lead with supplement-brand operations, not abstract martech language.
- Avoid centering “AI-powered” without explaining the workflow.
- Treat `PIM`, `DAM`, `partner portal`, and `syndication` as supporting categories inside one operating model.
- Use “product content operations” and “partner-ready product content” repeatedly as connective language.

## Metadata And Snippet Cleanup

These are important because current search snippets still show older messaging on the live site.

### Clean up immediately

- Remove old “sports supplement industry” phrasing where it still exists in metadata and indexable page copy.
- Keep sitewide metadata aligned to:
  - supplement brands
  - product content operations
  - unified PIM + DAM
  - AI localization
  - partner portals / syndication

### Watch for index lag

- The live domain may continue showing older titles and snippets until recrawled.
- Updating metadata alone is not enough if body copy and internal links still reinforce older positioning.
